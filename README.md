# Goose Universal Capsule

Goose Universal Capsule 是一个面向 Minecraft 1.20.1 Forge 的通用胶囊模组，提供结构扫描、封装与放置能力，并兼容多方块结构与实体内容。

## 功能

- 扫描指定区域，将方块、方块实体和可捕获实体写入胶囊模板
- 放置模板时支持旋转、阻挡检测和多方块展开
- 提供客户端预览、快捷键交互和 HUD 提示
- 支持模板保存、读取、重命名、样式切换与发放

## 依赖

- Minecraft 1.20.1
- Forge 47.3.0+

## 配置

配置目录位于：

```text
.minecraft/config/goose/goose-universalcapsule/
```

## 构建

```powershell
git clone https://github.com/LuckBigGoose/goose-universalcapsule.git
cd goose-universalcapsule
.\gradlew.bat build
```
